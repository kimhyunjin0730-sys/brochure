01
도킹 인증
태블릿을 Vault에 물리 결합 → PUF 키 검증 → 암호화 채널 활성화. 인증된 기기만 접근 가능.
02
데이터 접근
AES-256 복호화로 데이터 열람·편집. 모든 이력 감사 로그 자동 기록. 5단계 권한에 따라 접근 범위 제어.
03
도킹 해제
채널 즉시 종료. 태블릿 메모리 자동 소거. 모든 접근 권한 차단. 데이터는 Vault 내 잠금 상태 유지.
04
탈취·분해 시
PUF 보안키 즉시 자동 소멸. 저장 매체를 물리적으로 분리해도 복호화 절대 불가. 데이터 영구 보호.
